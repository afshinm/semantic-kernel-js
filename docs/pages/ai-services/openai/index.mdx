## OpenAI Overview

You can use the OpenAI provider to integrate with OpenAI's API for various AI capabilities such as text generation, embeddings, and more.

The Semantic Kernel [OpenAI Provider](https://www.npmjs.com/package/@semantic-kernel/openai) (`@semantic-kernel/openai`) is a package that allows you to interact with OpenAI's services seamlessly within the Semantic Kernel framework.
